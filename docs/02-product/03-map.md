# Product Roadmap

**Status** : Draft
**Version** : 1.0

---

# Vision

Le développement de ComLabs IAM est organisé en versions successives.

Chaque version apporte de nouvelles capacités sans remettre en cause le modèle métier existant.

---

# Version 1

Objectif :

Construire un moteur IAM robuste.

Fonctionnalités :

- Identity
- Login Identifier
- Password Credential
- Authentication
- Session
- JWT
- Roles
- Permissions
- Security Policies
- API REST

---

# Version 2

Objectif :

Moderniser l'authentification.

Fonctionnalités :

- OpenID Connect
- OAuth2
- Refresh Token
- MFA
- Audit avancé

---

# Version 3

Objectif :

Préparer les nouvelles méthodes d'authentification.

Fonctionnalités :

- Passkeys
- WebAuthn
- OTP
- Certificats
- Federation

---

# Long terme

Le produit devra rester compatible avec les futurs standards du marché.

Les évolutions devront préserver :

- le modèle métier ;
- la documentation ;
- l'architecture.
